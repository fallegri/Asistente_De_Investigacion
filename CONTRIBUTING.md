# Contribuir al Asistente de Investigación

¡Gracias por tu interés en mejorar este proyecto!

## Cómo contribuir

1. Haz un **fork** del repositorio
2. Crea una rama para tu mejora: `git checkout -b feature/mi-mejora`
3. Haz tus cambios y commitea: `git commit -m "feat: descripción"`
4. Sube tu rama: `git push origin feature/mi-mejora`
5. Abre un **Pull Request** describiendo qué cambiaste y por qué

## Ideas de mejoras bienvenidas

- Soporte para más metodologías de investigación
- Adaptación a otros formatos de perfil (no solo MDG100)
- Traducción a otros idiomas
- Mejoras en la validación de verbos Bloom
- Integración con otros modelos de IA
- Mejoras de accesibilidad y responsive

## Requisitos del entorno

Ver `README.md` sección de instalación.

## Código de conducta

Este proyecto sigue un ambiente abierto y respetuoso.
Cualquier contribución es bienvenida independientemente del nivel de experiencia.
